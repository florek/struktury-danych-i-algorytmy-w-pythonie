# Linear Search (Wyszukiwanie liniowe)

## Definicja

**Wyszukiwanie liniowe** (linear search) to najprostszy algorytm wyszukiwania elementu w liście.  
Polega na **sprawdzaniu kolejnych elementów jeden po drugim**, aż do znalezienia wartości lub końca listy.

---

## Implementacja

```python
def search_linear(value, list):
    idx = 0
    found = False
    while not found and idx < len(list):
        if list[idx] == value:
            found = True
        else:
            idx += 1
    return idx if idx < len(list) else None
```

### Jak to działa mentalnie

- sprawdzasz elementy od początku do końca
- każdy element porównujesz z szukaną wartością
- zatrzymujesz się gdy znajdziesz lub skończy się lista

### Właściwości

* czas: **O(n)**
* działa na listach posortowanych i nieposortowanych
* prosty w implementacji
* zawsze sprawdza wszystkie elementy w najgorszym przypadku

### Kiedy używać

* małe listy
* nieposortowane dane
* rzadkie wyszukiwania
* prostota jest ważniejsza niż wydajność

### Przykład wykonania krok po kroku

Wejściowa lista: `[1, 7, 3, 1, 2, 3, 5, 6]`, szukamy wartości: `5`

**Iteracja 1:**
- `idx = 0`, `found = False`
- `list[0] = 1 == 5?` → NIE
- `idx = 1`

**Iteracja 2:**
- `list[1] = 7 == 5?` → NIE
- `idx = 2`

**Iteracja 3:**
- `list[2] = 3 == 5?` → NIE
- `idx = 3`

**Iteracja 4:**
- `list[3] = 1 == 5?` → NIE
- `idx = 4`

**Iteracja 5:**
- `list[4] = 2 == 5?` → NIE
- `idx = 5`

**Iteracja 6:**
- `list[5] = 3 == 5?` → NIE
- `idx = 6`

**Iteracja 7:**
- `list[6] = 5 == 5?` → TAK → `found = True`
- Zwracamy `idx = 6`

Wynik: `6` (indeks wartości 5)

**Przykład 2: szukamy wartości, której nie ma**

Wejściowa lista: `[1, 7, 3, 1, 2, 3, 5, 6]`, szukamy wartości: `111`

**Iteracja 1-8:**
- Sprawdzamy wszystkie elementy: `1, 7, 3, 1, 2, 3, 5, 6`
- Żaden nie jest równy `111`
- `idx = 8`, `idx >= len(list)` → `idx < len(list)?` → NIE
- Zwracamy `None`

Wynik: `None` (wartość nie znaleziona)
