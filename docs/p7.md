# Binary Search (Wyszukiwanie binarne)

## Definicja

**Wyszukiwanie binarne** (binary search) to algorytm wyszukiwania elementu w **posortowanej liście**.  
Polega na **dzieleniu przestrzeni wyszukiwania na pół** w każdej iteracji, co znacznie przyspiesza proces.

---

## Implementacja

```python
def search_binary(value, list):
    start = 0
    end = len(list) - 1
    found = False
    while start <= end and not found:
        mid = (start + end) // 2
        if list[mid] == value:
            found = True
        else:
            if value < list[mid]:
                end = mid - 1
            else:
                start = mid + 1
    if found:
        return mid
    else:
        return None
```

### Jak to działa mentalnie

- lista musi być posortowana
- sprawdzasz element w środku przedziału
- jeśli wartość jest mniejsza, szukasz w lewej połowie
- jeśli większa, szukasz w prawej połowie
- zawężasz przedział do momentu znalezienia lub wyczerpania opcji

### Właściwości

* czas: **O(log n)**
* wymaga posortowanej listy
* bardzo szybki dla dużych zbiorów danych
* znacznie szybszy niż wyszukiwanie liniowe

### Kiedy używać

* lista jest posortowana
* potrzebujesz szybkiego wyszukiwania
* często wyszukujesz elementy w dużych zbiorach danych
* sortowanie raz + wiele wyszukiwań binarnych często lepsze niż wiele wyszukiwań liniowych
